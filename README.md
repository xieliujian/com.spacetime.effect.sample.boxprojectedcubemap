# Box Projected Cubemap 示例

[com.spacetime.effect](https://github.com/xieliujian/com.spacetime.effect) 的示例工程，演示**盒投影环境 Cubemap**（矩形房间内更准确的反射）。

## WebGL 演示

- **[在线体验](https://xieliujian.github.io/xieliujian_page/spacetime/effect/boxprojectedcubemap/)**

## 本地运行

- Unity **2022.3 LTS**（推荐与 `ProjectSettings/ProjectVersion.txt` 一致），**URP**。
- 打开场景：`Assets/Scenes/BoxProjectedCubemap.unity`。
- 内嵌子模块：[com.spacetime.core](https://github.com/xieliujian/com.spacetime.core)、[com.spacetime.effect](https://github.com/xieliujian/com.spacetime.effect)（`Packages/` 下，克隆需带 `--recurse-submodules`）。
